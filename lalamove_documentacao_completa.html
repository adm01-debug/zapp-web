# Algoritmo de assinatura HMAC-SHA256 (Python)
import hmac, hashlib, time, base64

def sign_request(method, path, body, api_key, secret_key):
    ts = str(int(time.time() * 1000))
    raw = f"{ts}
{method}
{path}

{body}"
    sig = hmac.new(secret_key.encode(), raw.encode(), hashlib.sha256).hexdigest()
    return f"hmac {api_key}:{ts}:{sig}"

headers = {
    "Authorization": sign_request(...),

3.3 Criptografia de Senha (RSA-OAEP)

// Algoritmo usado pelo app (Web Crypto API)
async function encryptPassword(password, publicKeyPem) {
  const nonce = generateNonce();
  const plaintext = `${nonce}@${password}`;
  const publicKey = await crypto.subtle.importKey(
    "spki", pemToDer(publicKeyPem),
    { name: "RSA-OAEP", hash: "SHA-256" }, false, ["encrypt"]
  );
  const ciphertext = await crypto.subtle.encrypt(
    { name: "RSA-OAEP" }, publicKey, new TextEncoder().encode(plaintext)
  );
  return `encrypted:${base64url(ciphertext)}`;
}
// Enviado como: password=encrypted:BASE64URL_CIFRADO
// Em scripts Python via curl: senha em texto plano funciona como fallback

⚙️5. UAPI Interna (br-uapi)

API PHP monolítica que expõe todas as funcionalidades do app web/mobile. Roteada por ?_m={method} no query string.

Formato de Resposta Padrão

{
  "ret": 0,       // 0 = sucesso, >0 = erro
  "msg": "",       // mensagem de erro (vazio se ret=0)
  "data": { ... }  // payload da resposta
}

🛡️11. WAF / Gateway de Segurança Argus

O Argus SDK é uma biblioteca JavaScript proprietária anti-bot. Ativo no Brasil: security_sdk_flag=1, argus_enabled=true.

⛔ Endpoints bloqueados sem token Argus: order_request, price_calculate, order_detail, order_cancel, order_edit
Resposta WAF: {"ret": 10010, "msg": "Gateway signature error"}

O SDK coleta: canvas fingerprint, WebGL, navigator.plugins, timing, mouse patterns. Impossível de replicar fora de um browser real sem acesso ao SDK proprietário.

✅ Solução: Usar Playwright com browser real (Chrome/Chromium) — o browser executa o Argus SDK nativamente e gera os tokens automaticamente.

🔀13. Ambiente de Execução e Proxy

⚠️ O container tem proxy configurado via HTTP_PROXY/HTTPS_PROXY. Ferramentas nativas Python (requests, urllib) não passam corretamente. Use curl via subprocess.

Workaround — curl via Python

import subprocess, json, time, random

def uapi_call(method, args=None, token=""):
    ts = int(time.time())
    su = f"{int(time.time()*1000)}ehll{int(random.random()*1e10)}"
    result = subprocess.run([
        'curl', '-s', '-m', '15', '--compressed', '-X', 'POST',
        '-H', 'User-Agent: Mozilla/5.0 AppleWebKit/537.36',
        '-H', 'Content-Type: application/x-www-form-urlencoded; charset=utf-8',
        '-H', 'Origin: https://web.lalamove.com',
        '-H', 'X-LLM-LOCATION: BR',
        '--data-urlencode', f'token={token}',
        '--data-urlencode', 'is_ep=1',
        '--data-urlencode', 'hcountry=20000',
        '--data-urlencode', 'hlang=pt_BR',
        '--data-urlencode', 'city_id=21001',
        '--data-urlencode', f'args={json.dumps(args or {})}',
        f'https://br-uapi.lalamove.com/index.php?_m={method}&hcountry=20000&_su={su}&_t={ts}&device_id=playwright-001&version=5.37.0&revision=53700&os=web&device_type=web'
    ], capture_output=True)
    return json.loads(result.stdout.decode(errors='replace'))

⚠️15. ERRATA — Correções à Documentação Oficial

Cada correção abaixo foi comprovada com output direto da API de produção Brasil. A documentação oficial Lalamove contém erros de campo que só surgem em testes reais.
CORREÇÃO 1: Campo services (não serviceTypes)

Documentação oficial afirma que o objeto city usa campo serviceTypes

❌ ERRADO: city.serviceTypes = ["MOTORCYCLE", ...]
✅ CORRETO: city.services = ["MOTORCYCLE", "CAR", "MPV", "VAN", "TRUCK175", "TRUCK330"]

💥 Impacto: Código que acessa city.serviceTypes retorna undefined.

CORREÇÃO 2: driverId é sempre string, nunca null

Documentação oficial afirma que driverId pode ser null quando não atribuído

❌ ERRADO: { "driverId": null }
✅ CORRETO: { "driverId": "" } ← string vazia, NUNCA null

💥 Impacto: Verificação if (driverId === null) falha. Deve ser if (!driverId).

CORREÇÃO 3: cancelParty e cancelledReason não existem no Brasil

Documentação oficial lista esses campos na resposta de cancelamento

❌ ERRADO: Response inclui cancelParty e cancelledReason
✅ CORRETO: Esses campos NÃO estão presentes na API Brasil (confirmado em produção)

💥 Impacto: Código que acessa esses campos retorna undefined, causando erros silenciosos.
